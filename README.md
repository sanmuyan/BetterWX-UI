## ❓ 常见问题解决方案

- **请先按流程逐个试一遍：**
- 1.先关闭所有微信程序，再使用。
- 2.使用管理员权限启动。
- 3.放到微信目录内启动。
- 4.清除 BetterWX-UI 缓存，重启软件。
- 5.删除微信文件夹，使用官方微信客户端覆盖安装一次。

- **程序闪退：**
- 请安装webview2运行环境
- Win7请同时安装VxKex插件

- **环境异常：**
- 删除共存重建

- [webview2下载地址](https://developer.microsoft.com/zh-cn/microsoft-edge/webview2/?form=MA13LH#download)
